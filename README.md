# Alien Adventures
Project 4 for MSCH-C220

A basic 2D platformer. 

How to play: press A and D to move the character and the Space bar to jump. 

## Implementation

Built with Godot 4.1

Features: 

Double Jump

Multiple Levels

Main, pause, and end menus
